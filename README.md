# Birthday Cake Skill

A reusable Codex skill for creating cinematic interactive birthday webpages with React, TypeScript, Three.js, React Three Fiber, candle blowing interactions, draggable 3D Polaroid memory walls, Vercel deployment, and QR-code sharing.

## Install in Codex

Use the skill installer with this repository path:

```text
$skill-installer install https://github.com/Mirandacao82/birthdaycake/tree/main/skills/birthday-3d-memory-web
```

Then restart Codex so the skill is discovered.

## Use

```text
$birthday-3d-memory-web Create an online birthday surprise page with a 3D cake and photo memory wall.
```

Or use natural language:

```text
Use the birthday 3D memory web skill to create a shareable birthday webpage from my photo folder.
```

## What it covers

- Vite + React 19 + TypeScript scaffolding
- React Three Fiber scene architecture
- Candle blowing state machine
- 3D cake, flames, smoke, balloons, confetti
- Draggable photo memory wall and lightbox
- Photo thumbnail optimization for deployment
- Vercel public URL and QR-code sharing
- Troubleshooting for online R3F/Suspense rendering issues
